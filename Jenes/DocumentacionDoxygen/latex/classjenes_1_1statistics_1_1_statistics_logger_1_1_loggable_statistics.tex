\subsection{Detailed Description}
This class provides the support for making a statistics loggable.

It is simply required to make the own statistics class extending \hyperlink{classjenes_1_1statistics_1_1_statistics_logger_1_1_loggable_statistics}{Loggable\-Statistics}.
