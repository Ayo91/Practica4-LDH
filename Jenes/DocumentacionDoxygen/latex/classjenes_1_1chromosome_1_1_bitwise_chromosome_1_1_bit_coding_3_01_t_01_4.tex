\subsection{Detailed Description}
Represents the coding used by a \hyperlink{classjenes_1_1chromosome_1_1_bitwise_chromosome}{Bitwise\-Chromosome}. A Bit\-Coding can be used by subclassing this class and providing implementation for \hyperlink{}{Bit\-Coding\#encode(\-Object)} and \hyperlink{classjenes_1_1chromosome_1_1_bitwise_chromosome_1_1_bit_coding_3_01_t_01_4_ac07b8fe9be73bd1f2111301eb5fee6cf}{decode(int)} methods. A \hyperlink{enumjenes_1_1chromosome_1_1_bitwise_chromosome_1_1_bit_size}{Bit\-Size} object is useful to specify the number of bits to use foreach coded number.
