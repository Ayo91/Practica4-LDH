Detailed Description

This class provides an utility wrapper for logging the statistics.

In order to log statistics, the class has to extend LoggableStatistics in order to automatically retrieve figures. Figures are provided by methods returning double value figures and annotated as .

For example: 

class MyStatistics extends LoggableStatistics {
   ...
   &#064;Loggable("SomeFigure")
   double getSomeValue() { ... }
   ...
   &#064;Loggable("SomeCount")
   double getSomeCount() { ... }
   ...
}

Statistics are passed to the logger as specified at instantiation time.
