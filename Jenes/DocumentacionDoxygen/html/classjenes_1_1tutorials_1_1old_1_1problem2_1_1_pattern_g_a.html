Detailed Description

Tutorial showing how to extend GeneticAlgorithm and how to use the flexible and configurable breeding structure in Jenes. The problem consists in searching a pattern of integers with a given precision. Solutions flow through two different crossovers in parallel. Some are processed by a single point crossover, the other by a double point crossover. After solutions are mutated.

This class implements the algorithm by extending GeneticAlgorithm.
