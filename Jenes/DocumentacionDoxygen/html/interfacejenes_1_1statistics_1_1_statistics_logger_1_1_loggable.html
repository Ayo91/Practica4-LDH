String label ()
 

Detailed Description

Annotation for methods providing some figure in a LoggableStatistics object.

The method has to be double foo(). Requires to specify the figure's label. The method name and annotation label are not necessarily related. For example:

&#064;Loggable("SomeStatistics")
double getSomeStatistic() {...}
